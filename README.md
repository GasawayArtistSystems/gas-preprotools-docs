# GAS PrePro Tools — Documentation Site

Built with [Docusaurus](https://docusaurus.io/). Deploys automatically to GitHub Pages.

## Local development

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Structure

```
docs/
├── getting-started/
├── script-tab/
├── blocking/
├── shot-design/
├── build-and-watch/
├── project-settings/
└── troubleshooting/
```

## Deployment

Push to `main` → GitHub Actions builds and deploys automatically.

Live site: https://GasawayArtistSystems.github.io/gas-preprotools-docs/
