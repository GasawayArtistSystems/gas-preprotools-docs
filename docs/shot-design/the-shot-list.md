# The Shot List

The shot list shows every shot for the current scene. It lives in the left panel of Director View when in Shot Edit mode, and also appears as a column in the Script Tab.

## Shot list columns

| Column | Description |
|--------|-------------|
| Shot # | Shot number within the scene (A, B, C…) |
| Type | Shot size (ECU, CU, MCU, MS, WS, etc.) |
| Camera | Assigned camera actor |
| Length | Shot length in eighths of a page |
| Status | Build status dot |

## Adding a shot

Click **Add Shot** in the shot list toolbar. A new shot row is created. Set the shot type and assign a camera.

## Reordering shots

Drag shot rows up or down to reorder them within the scene.

## Deleting a shot

Select a shot row and press **Delete**, or right-click and choose **Remove Shot**.

## Shot numbering

Shots are numbered within each scene: Scene 10 shots become `10A`, `10B`, `10C`, etc. Numbering updates automatically when you add or reorder shots.

## Status dot

The colored dot on each scene in the scene list reflects the state of its shots. See [Understanding the Status Dot](./understanding-the-status-dot).
