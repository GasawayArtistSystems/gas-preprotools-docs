# Generating Page Breaks

Page breaks divide your script into pages and are used to calculate shot lengths in eighths of a page — the industry-standard unit for scheduling.

## Auto-generate page breaks

Click **Generate Page Breaks** in the Script Tab toolbar. GAS Tools will estimate page breaks based on Final Draft's standard page length (~53 lines per page).

:::note
Auto-generation is available only when no page breaks exist yet. If you already have page breaks, clear them first.
:::

## Adjusting page breaks

Hold **Alt** and drag any page break marker up or down to reposition it. The marker snaps to paragraph boundaries.

## Deleting a page break

Hold **Alt** and click a page break marker to delete it. The first page break (PAGE 2) cannot be deleted.

## Why page breaks matter

Shot lengths displayed in the Shot List are calculated in **eighths of a page**. Accurate page breaks = accurate eighths. This directly feeds your breakdown and scheduling data.
