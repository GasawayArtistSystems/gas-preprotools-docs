# Editing the Script

After importing, you can edit your script directly inside UE without going back to Final Draft.

## Entering edit mode

Click the **Edit** button in the Script Tab toolbar to toggle edit mode on. The script panel will indicate you're in edit mode.

## Editing text

Double-click any paragraph to open an inline text editor. Make your changes and click away or press **Enter** to confirm.

## Changing block types

Right-click any paragraph to open the block type menu. You can change a block from Action to Dialogue, or from Character to Scene Heading, etc.

## Adding blocks

With edit mode active, hover between two paragraphs to reveal the **+** insert gutter. Click it to add a new block above or below.

## Undo

GAS Tools supports multi-step undo within the script editor. Press **Ctrl+Z** to step back through changes.

## Saving

Your script is saved automatically when you save the UE project (**Ctrl+S**). A dirty indicator (●) in the Script Tab title means there are unsaved changes.

:::note
Script edits are stored in the GAS project asset, not written back to the original `.fdx` file.
:::
