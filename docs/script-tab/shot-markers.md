# Shot Markers

Shot markers overlay the script panel to show where each shot begins and ends relative to the script text.

## What shot markers show

Each shot marker appears as a colored pill in the right gutter of the script panel. The pill spans the lines of script the shot covers.

- **Pill color** indicates shot type (CU, MCU, WS, etc.)
- **Pill height** corresponds to the script lines the shot covers
- **Tooltip** on hover shows shot name and length in eighths

## Creating shots from the script panel

In the Script Tab, you can draw a shot range directly on the script:

1. Hold **Alt** and click-drag across the lines you want the shot to cover.
2. A shot creation dialog will appear.
3. Assign a shot type and camera.

## Moving shots

Hold **Alt** and drag an existing pill up or down to reposition a shot's start point.

## Resizing shots

Hold **Alt** and drag the bottom edge of a pill to extend or shorten its tail.

## Shot list

All shots across all scenes are also visible in the **Shot List** panel in the Director View. See [The Shot List](../shot-design/the-shot-list).
