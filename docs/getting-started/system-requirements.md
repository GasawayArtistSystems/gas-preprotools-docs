# System Requirements

## Unreal Engine

- **Unreal Engine 5.6** or later (required)

## Operating System

- Windows 10 or Windows 11 (64-bit)

## Hardware

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| CPU | Intel i7 / AMD Ryzen 7 | Intel i9 / AMD Ryzen 9 |
| RAM | 16 GB | 32 GB |
| GPU | NVIDIA GTX 1080 / AMD RX 5700 | NVIDIA RTX 3080 or better |
| Storage | 10 GB free | SSD with 50 GB free |

## Software

- [Final Draft](https://www.finaldraft.com/) (to export `.fdx` files for import)
- Visual Studio 2022 (if building from source)

:::note
GAS PrePro Tools is a C++ plugin. Your UE project must support C++ — Blueprint-only projects will need to be converted before installation.
:::
