# What is GAS PrePro Tools?

GAS PrePro Tools is an Unreal Engine 5 plugin built for film and TV directors. It lets you go from a Final Draft script to a fully pre-visualized scene — inside Unreal Engine, without needing to be a technical artist.

## What it does

- **Import scripts** directly from Final Draft (.fdx) into Unreal Engine
- **Edit your script** inside the engine — adjust text, scene headings, and page breaks without leaving UE
- **Block scenes in 3D** — place and animate your cast in a blocking level
- **Design shots** — create shot markers, assign cameras, and build your shot list
- **Build scene sequences** — compile your blocking and shots into a pre-vis sequence
- **Watch pre-vis** — play back your sequences in the UE viewport

## Who it's for

GAS PrePro Tools is designed for directors and pre-production teams who want to visualize their film before a single frame is shot. No deep Unreal Engine expertise required.

## The workflow

```
Import Script → Edit Script → Block Scene → Design Shots → Build Sequence → Watch Pre-vis
```

Each step in this guide corresponds to a section of the plugin.
