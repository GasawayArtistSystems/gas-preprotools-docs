# Project Preferences

Project Preferences let you configure global settings for your GAS Tools project. Open them by clicking the **gear icon (⚙️)** in the GAS Tools toolbar.

## Project name

Sets the display name for your project inside the plugin.

## Aspect ratio

Sets the target aspect ratio for your pre-vis. This affects:

- Viewport framing guides
- Render output resolution

Common choices:

| Format | Ratio |
|--------|-------|
| Widescreen film | 2.39:1 |
| Widescreen HD | 16:9 |
| Academy | 1.85:1 |
| Square | 1:1 |

## Render quality

Sets the quality preset used when rendering frames from the plugin:

- **Preview** — fast, lower quality, for quick review
- **Standard** — balanced quality for sharing
- **Final** — high quality for deliverables

## Render format

Sets the output format for rendered frames:

- **PNG** — lossless, larger files
- **EXR** — high dynamic range, for compositing

## Render output path

Renders are saved to `Saved/MovieRenders/` inside your UE project folder by default.
